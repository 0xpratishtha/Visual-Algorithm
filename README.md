# Algorithmic Visualization Hub

An interactive web application that visualizes how common algorithms work in real time, built to make learning algorithms intuitive and engaging.

## Features

- **Sorting Algorithms** — Bubble Sort, Selection Sort, Quick Sort with animated bar visualizations
- **Pathfinding Algorithms** — Dijkstra's Algorithm on an interactive grid
- **Searching Algorithms** — Binary Search with step-by-step visualization

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Tailwind CSS

## How to Use

1. Clone or download the repository
2. Open `index.html` in any browser
3. Navigate to Sorting, Pathfinding, or Searching from the navbar
4. Select an algorithm, adjust speed, and hit visualize!

## Purpose

Built as a college project to strengthen understanding of core computer science algorithms through visual, interactive demonstrations.
